📂 Registros Salvos

Estes registros estão salvos no localStorage do seu navegador — ficam neste dispositivo e somem se limpar os dados do navegador. No SharePoint, ficam em uma lista acessível por qualquer pessoa da empresa.

☁️ Como funciona no SharePoint?

O SharePoint é como uma "pasta na nuvem" da sua empresa dentro do Microsoft 365. Você cria uma Lista (parecida com uma tabela do Excel) e cada notificação vira uma linha nessa lista.

O que acontece quando alguém envia o formulário:

  1. O JavaScript do formulário monta um "pacote de dados" com todos os campos preenchidos.
  2. Esse pacote é enviado via API REST (uma chamada de internet) direto para a lista do SharePoint.
  3. O SharePoint confirma que salvou e o formulário mostra a tela de sucesso com o protocolo.
  4. Qualquer pessoa com acesso à lista no SharePoint vê o registro em tempo real.

Para ativar: altere persistencia: 'local' para persistencia: 'sharepoint' no bloco CONFIG do arquivo e preencha a URL do seu site.
